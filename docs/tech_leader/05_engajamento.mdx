# Engajamento

Quatro Regras para Engajar o seu time:

1. Use e abuse dos 3P's:

:::tip
Propósito, Processos e Pessoas!
:::

- Traga o propósito para sua equipe;
- Diga sempre o porquê você precisa de uma determinada coisa;
- Saiba o porquê de cada liderado;

2. Não tenha medo de ter conversas difíceis;

:::tip
Líderes melhores são aqueles que estão preparados para conversas difíceis;
:::

3. Dividir as decisões:

:::tip
Analise seu cenário para tomar a melhor decisão!
:::

- irreversíveis: não pode ser revisto;
- reversíveis: pode ser coletivo, pode ser mais experimentado;

4. Domine as first layers skills;

- softskills:
    - adaptabilidade e inteligência emocional;
    - colaboração e influência;
    - liderança e gestão de pessoas;

- habilidades empreendedoras:
    - criatividade e resolução de problemas;
    - inovação;
    - fazer acontecer / protagonismo;

- Preparação para o mercado de trabalho;
    - Produtividade;
    - comunicação /inglês;
    - negociação e vendas;

- habilidades técnicas: específicas para cada área de atuação;

:::caution
Focar no tronco da árvore, ou seja, nas habilidades primárias para potencializar as habilidades técnicas;
:::
